JavaScript Event Delegation 事件委托

什么叫委托呢？想想我们现实生活中，自己要干的事，让别人来帮忙完成，这就是把事情“委托”给别人。
Javascript的事件委托机制同理:
本来一个监听函数，要处理子节点触发的事件，现在，把这个监听函数绑定到父层节点上，让父辈来监听事件。在父辈元素的监听函数中，可通过event.target属性拿到触发事件的原始元素（子节点），然后再对其进行处理。
